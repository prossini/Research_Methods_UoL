NOTES

Non-internet households were set to 2=No for SNSUSE and ON7_a-h.

The Wave 74 dataset contains coded open-end responses for up to three mentions for question REDRACIALINEQOTH. Those coded open-end responses are contained in variables:
REDRACIALINEQOTH_M1
REDRACIALINEQOTH_M2
REDRACIALINEQOTH_M3

Please see accompanying PDF readme file for additional details for analyzing data on online harassment. 

For a small number of respondents with high risk of identification, certain values have been randomly swapped with those of lower risk cases with similar characteristics.

***************************************************************************************************************************
WEIGHTS 

The Wave 74 dataset contains three weights: 

WEIGHT_W74 is the weight for the sample. Data for most Pew Research Center reports are analyzed using this weight.
WEIGHT_W43_W74 is a longitudinal weight for respondents to both Wave 43 and Wave 74. This weight should be used for longitudinal anaysis for questions RACATTN, RACESURV5, RACESURV6, RACESURV12, RACESURV13 and RACESURV28.
WEIGHT_W67_W74 is a longitudinal weight for respondents to both Wave 67 and Wave 74. This weight should be used for longitudinal anaysis for question COVID_SCI6E.
